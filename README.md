# 🚀 Employee Burnout Detection System

## 📌 Overview

The **Employee Burnout Detection System** is an intelligent application designed to monitor and analyze user typing behavior to detect stress and burnout levels in real-time.

This project combines **frontend visualization (React + Vite + TypeScript)** with a **backend machine learning model (Python)** to provide actionable insights into employee mental health.

---

## 🎯 Problem Statement

Employee burnout is a growing issue in modern workplaces, leading to:

* Reduced productivity
* Increased absenteeism
* Mental health challenges

Traditional methods fail to provide **real-time detection**. This project aims to solve that gap using behavioral data.

---

## 💡 Solution

This system tracks typing patterns such as:

* Typing speed
* Keystroke dynamics
* Interaction behavior

These inputs are processed using a **Python-based stress prediction model**, which classifies stress levels and displays them visually.

## 🧠 How It Works

1. User starts typing in the application
2. System captures typing speed & behavior
3. Data is sent to the Python model
4. Model predicts stress level
5. UI updates the stress indicator dynamically

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/pinkyrani1430-cell/Employee-burnout-detection-
cd employee-burnout-detection
```

### 2️⃣ Install Frontend Dependencies

```bash
npm install
```

### 3️⃣ Run Frontend

```bash
npm run dev
```

### 4️⃣ Run Python Model

